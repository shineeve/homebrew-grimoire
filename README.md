# Shineeve Grimoire

## How do I install these formulae?

`brew install shineeve/grimoire/<formula>`

Or `brew tap shineeve/grimoire` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "shineeve/grimoire"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
